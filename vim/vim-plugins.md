# Vim-Plugins
## Surround Plugin
1. surround a word in "a"
- select the word and use command `ysiw"`

1. change " to '
- cs"'

1. change ' to tag <q>
- cs'<q>


1. remove tag <q>
- dst

1. remove quote ""
- ds"

1. apply changes to whole line
- yss
